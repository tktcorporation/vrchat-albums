# 写真ピックアップ機能 設計書

作成日: 2026-03-09

## 概要

SNS 投稿候補の写真を一時的にストックするピックアップ機能。お気に入りより賞味期限が短い「投稿候補」を管理する。

## 背景・目的

VRChat で撮影した写真の中から SNS に投稿したいものを選んでおく仕組みが必要。お気に入り（Favorites）は恒久的な保存が目的だが、ピックアップは「近々投稿する予定の写真」を一時的にストックする用途で使用する。

## 設計方針

- ユーザーが手動でクリアするまで保持する（自動削除しない）
- 写真は photoId で識別する（VRChatPhoto モデルの主キー）
- 追加順を保持するため createdAt でソートする

## データモデル

### PhotoPickups テーブル

| カラム | 型 | 説明 |
|--------|-----|------|
| id | UUID | 主キー（uuidv7） |
| photoId | STRING | 写真 ID（ユニーク制約） |
| createdAt | DATE | 追加日時 |
| updatedAt | DATE | 更新日時 |

## UI フロー

1. 写真一覧でドラッグ or コンテキストメニューからピックアップに追加
2. ドロップゾーンにドロップしてピックアップ登録
3. ピックアップ一覧ダイアログで確認・コピー・クリア

## 削除条件

- 全クリアボタンで全件削除
- 個別削除はコンテキストメニューから実行
